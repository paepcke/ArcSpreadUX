The Workspace Window

  • Scrollwheel advances/rewinds screens NOTE: Number of screens advanced per notch is set in the Windows mouse control panel.
  • Shift-arrow keys work as in Windows File Explorer's Thumbnails view: selection.
  • Image size adjustment: vertical slider
  • Images per row adjustment: horizontal slider
  • Reset workspace size: drag images-per-row slider to 0 and let go.
  • Zoom button (cnt-=/cnt-plus/cnt-minus): new zoom windows for each selected image.
  • Can drag from Workspace window.
