Keyboard Shortcuts

Editor Window

Purpose: save on frequent drag/drop actions, replacing them with a keyboard shortcut. You will define a key stroke and a destination cell in this editor. Later, you can select items in other cells or in the workspace, and copy them to the destination cell by entering the keystroke. You can define as many of these shortcuts as you like.
  • Note: : Changes are not saved automatically. Use buttons.
  • Move among fields: TAB/Enter.
  • More rows added automatically when all rows are full.
  • Clear current, or all selected rows: DEL
  • Remove current, or all selected rows: numpad's DEL key.
  • Select all: cnt-a
  • Close window immediately: cnt-w.
