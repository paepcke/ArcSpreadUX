The Metadata Editor Window

  • Purpose: add attribute/value metadata to one item.
  • Getting there: double-click on photo in Workspace or sheet.
  • Note: : Changes not saved automatically. Use buttons.
  • Note: : Edit fields that start with '@' at your own peril!
  • Move among fields: TAB/Enter.
  • More rows added automatically when all rows are full.
  • Clear current, or all selected rows: DEL
  • Remove current, or all selected rows: numpad's DEL key.
  • Select all: cnt-a
  • Close window immediately: cnt-w.
