The Sheet Window

  • Load photos: Select cell; Context menu->Load object files...
  • Image formats: .jpg, .png, .gif.
  • See cell detail: click on cell; -> Workspace window updates.
  • Edit metadata: double-click image.
  • Dragging: select first. Then drag.
  • Delete image from this worksheet (not from disk): drag to cell 1,0)
  • Can drag to MS File Explorer, Desktop, Photoshop
  • Can drag from MS File Explorer and Desktop
  • Enter text to cell: select cell. Then type. Enter when done.
  • Formula ex.: =A1[Person=Frank].
  • Save cell: Context menu.
  • Save whole sheet: File->Save sheet...
  • Clear cell: Context menu.
  • Resize row: drag in col 0.
  • Resize col: drag col header edges.
  • Restore default row size: double click col 0.
